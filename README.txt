Week 8 README

Purpose:
BeatBridge is a music discovery application that helps users find new songs and independent artists based on their mood and favorite 
genres. Users can also create playlists, save songs, and share playlists with others.

Project Contents:
- Project Contents:
- VB.NET Source Code: Contains the code for creating the BeatBridge project folders, saving project information, and reading/writing JSON data.
- App Development Document: Describes the BeatBridge app idea, user story, features, variables, and development plan.
- Change Log: Summarizes the improvements made to the project, including JSON integration, refactored procedures, and completed TODO items.
- Screenshot Proof: Provides proof that the application runs correctly, creates the required folder structure, and saves the Project.json file.
- README.txt: Explains the purpose of the project, its contents, and how to understand the submission.

How to Understand This Submission:
This project demonstrates the planning and development of the BeatBridge application. The folder structure organizes all project 
resources, while the Visual Basic application creates the project folders and saves project information as a JSON file using the 
Newtonsoft.Json package.
